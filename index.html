Introduction to Shell

Learn the fundamentals of the command line interface across macOS, Linux, and Windows. Master essential commands that work everywhere.

LS
Lucas de Souza
December 14, 2025
12 min read

The shell is a command-line interface that allows you to interact with your operating system using text commands. Whether you're on macOS, Linux, or Windows, understanding the shell is a fundamental skill for any developer, system administrator, or power user.

In this guide, we'll cover the essential commands that you'll use daily, with examples for all three major operating systems.

What is a Shell?

A shell is a program that takes commands from the keyboard and gives them to the operating system to perform. Different operating systems have different default shells:

  • macOS: zsh (Z shell) ‚Äî since Catalina, previously bash
  • Linux: bash (Bourne Again Shell) ‚Äî most common default
  • Windows: PowerShell

Let's start with the most basic command. Open your terminal and try navigating to your Desktop folder:

Changing Directories

All Platforms
Terminal 
cd Documents             # Enter the Documents folder
cd ..                    # Go back one directory (parent folder)
cd ../..                 # Go back two directories
cd ~                     # Go to home directory (macOS/Linux)
cd $HOME                 # Go to home directory (PowerShell)

Listing Files and Folders

üçé macOS üêß Linux
Terminal (macOS/Linux) 
ls                       # List files and folders
ls -l                    # Long format with details
ls -la                   # Include hidden files (start with .)
ls -lh                   # Human-readable file sizes
ls -lt                   # Sort by modification time
ü™ü Windows PowerShell
PowerShell 
ls                       # Alias for Get-ChildItem
dir                      # Also works
Get-ChildItem            # Full PowerShell command
Get-ChildItem -Force     # Include hidden files
ls -Hidden               # Show only hidden files

File Operations

Learn how to create, copy, move, and delete files safely from the command line.

Creating Files and Folders

üçé macOS üêß Linux
Terminal (macOS/Linux) 
mkdir my-folder          # Create a new folder
mkdir -p path/to/folder  # Create nested folders
touch myfile.txt         # Create an empty file
echo "Hello" > file.txt  # Create file with content
ü™ü Windows PowerShell
PowerShell 
mkdir my-folder          # Create a new folder
New-Item -ItemType Directory -Path "path\to\folder"
New-Item myfile.txt      # Create an empty file
"Hello" | Out-File file.txt  # Create file with content

Copying Files

üçé macOS üêß Linux
Terminal (macOS/Linux)

ü™ü Windows PowerShell
PowerShell

Copy-Item folder -Recurse backup # Copy folder recursively
